# Laporan Praktikum

Direktori ini berfungsi sebagai wadah untuk pengumpulan laporan praktikum sesuai dengan modul praktikum.

## Struktur Folder
Adapun untuk struktur laporan praktikum sebagai berikut:
```
Kelas(A)_Modul(4)_Ucky_12245/
├─ Dokumentasi/
│  └─ README.md
├─ Laporan Praktikum/
│  └─ README.md
├─ Source Code/
│  └─ README.md
├─ .gitignore
└─ README.md
```
Khusus untuk direktori Source Code, file ```README.md``` digunakan untuk menjelaskan Code yang telah Anda buat.
